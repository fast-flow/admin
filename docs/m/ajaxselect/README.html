prompt & prompt{value}

data-ajaxselect-prompt 和 data-ajaxselect-prompterror 的格式可以为

"error_msg|错误原因="
"error_msg|错误原因=管理员审核不通过"
"error_msg|错误原因=管理员审核不通过&degree|严重程度="
